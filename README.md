# skills

Hermes Agent skills published by snav.

## Install

```bash
hermes skills install simpolism/skills/astrology
```

## Skills

- **astrology** — Compute astrological charts (natal, synastry, transit) and Human Design bodygraphs from birth date/time/place. Returns dense, LLM-readable chart text via a hosted endpoint. Pure Python stdlib client — no Node, no install, OS-neutral.
